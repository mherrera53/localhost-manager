# Configuration Files

This directory contains configuration files for Localhost Manager.

## Files

### hosts.json
Main configuration file containing all virtual hosts grouped by categories.

**Structure:**
```json
{
  "groups": [
    {
      "id": "group-id",
      "name": "Group Name",
      "hosts": [
        {
          "domain": "example.local",
          "aliases": ["www.example.local"],
          "documentRoot": "/path/to/project",
          "phpVersion": "8.4",
          "ssl": true,
          "enabled": true
        }
      ]
    }
  ]
}
```

See `hosts.json.example` for a complete example.

### hosts.txt
Generated file containing entries for `/etc/hosts`.

This file is auto-generated by the application and should not be edited manually.

### vhosts.conf
Generated Apache virtual host configuration.

This file is auto-generated by the application and should not be edited manually.

## Setup

1. Copy `hosts.json.example` to `hosts.json`
2. Edit `hosts.json` with your local domains
3. Use the web interface or desktop app to manage your hosts
